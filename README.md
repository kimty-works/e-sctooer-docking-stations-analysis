# Seoul Shared E-Scooter Parking Demand Analysis  
Course Work(Data Theroy 2023-1, Chungang University)

## Project Introduction  
This project analyzes the mismatch between designated e-scooter parking zones and actual illegal parking occurrences in Seoul.  
By leveraging spatial data analysis, the project aims to propose optimal parking locations to improve urban safety and public space management.  

## Team Introduction  
- Taeyoung Kim : Problem definition, hypothesis setting, EDA, spatial analysis, clustering design  
- Team Members : Problem definition, Data preprocessing, EDA, spatial analysis, visualization

## Methodology  
1. Data Preprocessing  
   - Converted address data into geographic coordinates  
   - Cleaned and structured spatial datasets  

2. Exploratory Data Analysis (EDA)  
   - Analyzed patterns of illegal parking incidents  
   - Compared parking zone types and accident occurrences  

3. Spatial Analysis  
   - Visualized high-density areas using Folium MarkerCluster  
   - Identified spatial mismatch between parking zones and actual demand  

4. Clustering  
   - Designed multi-stage clustering approach  
   - Derived high-demand regions for new parking zone allocation  

## Main Features  
1. Spatial Visualization  
   - Implemented interactive map visualization using Folium  
   - Highlighted high-density illegal parking areas  

2. Demand Hotspot Identification  
   - Identified top high-density regions with frequent incidents  

3. Parking Optimization Strategy  
   - Proposed new parking locations based on spatial demand analysis  

## Key Insights  
- A significant portion of illegal parking occurs outside designated parking zones  
- Spatial mismatch exists between existing infrastructure and actual usage patterns  
- High-density illegal parking areas indicate unmet parking demand  

## Expected Impact  
- Improves urban mobility and pedestrian safety  
- Provides data-driven guidelines for parking zone planning  
- Supports efficient management of shared mobility services
